Plug-in features

  • Generates ODE makefiles for Java and C/C++ projects.
  • Builds Java and C/C++ projects.
  • Enables building of an existing sandbox in Eclipse environment.
  • Enables building of a source sub tree of an existing sandbox in Eclipse environment.
  • Enables viewing files in the sandbox chain.

Installing with the Install/Update Manager

The ODE Plug-in can be installed via the Install/Update feature of the workbench. The URL for the ODE Plug-in Update Manager site is:
http://w3.sdwb.ibm.com/plugins/sdwb/install
Following are the steps for installing the ODE plug-in using the Update Manager:
  • Start the workbench
  • Open the Update Manager perspective by selecting "Help > Software Updates > Update Manager" menu item. [see Figure]
  • In the "Feature Updates" view, right click and select "New > Site Bookmark..." menu item from the context menu. The "New Site Bookmark" dialog appears. [see Figure]
  • In the "Name" field enter a name you wish. In the "URL" field enter the above specified URL. Click the "Finish" button to create the bookmark. A tree node with the name entered in the "Name" field is created in the "Feature Updates" view. [see Figure]
  • In the "Feature Updates" view, select the feature you wish to install by expanding the nodes of the site tree created in the previous step. [see Figure]
  • In the "Preview" view, click the "Install" button. The "Feature Install" wizard will take you through the installation process.
  • After the installing the plugin and restart the workbench to start using the ODE Plug-in.
This plug-in has been tested with ODE 4.0 using WSWB 2.0.2 on the Windows 2000 platform. Although it should work with previous versions of ODE on all the platforms that ODE and Eclipse support, it is recommended to use a workbench based on Eclipse 2.0 or later.  If you use the CDT for C/C++ development and wish to use the ODE plugin with it, version 1.0.1 or higher is required.
