Step 2. Create a site bookmark for the SDWB download site

Before you can download the ODE Build Feature, you must create a site bookmark for the SDWB download site. If you have already created this bookmark for another SDWB feature install, go to Step 3. Install the ODE Build Feature.

To create a site bookmark:

  1. Right-click anywhere in the Feature Updates view, then click New > Site Bookmark.
    Figure 2. Creating a new Site Bookmark
  2. In the New Site Bookmark window:
    1. Type a site name of your choice, for example, SDWB 5.0 Features.
    2. Type the SDWB download site URL: http://w3.sdwb.ibm.com/plugins/sdwb/install/.
    3. Select Eclipse update site.
    4. Click Finish.
