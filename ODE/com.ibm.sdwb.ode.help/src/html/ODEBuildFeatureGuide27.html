Known limitations of the ODE Build Feature

The following limitations should be noted before using the ODE Build Feature on your project:

  • The path to the project workspace should not have any spaces in it.
  • There can be only one source folder in a project and it should be named src.
  • The src folder must be an immediate subdirectory of the project folder.
  • Java source files should be created in a Java package. The source files should not be located directly under the src folder.
