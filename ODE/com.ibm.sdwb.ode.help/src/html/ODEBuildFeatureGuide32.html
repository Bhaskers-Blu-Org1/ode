Using Workbench views to perform ODE Build Feature tasks

You can perform ODE Build Feature tasks using the following Workbench views:

  • Navigator View
  • Package Explorer view
  • C/C++ Project view

To show Workbench views in your workspace:

  • Click Window > Show View.
  • Select a view from the list or select Other for additional views.
  • Click OK to show the view in your workspace.
Note:
If these views are not available in the list of views in your workspace, you may have to open a specialized perspective first. Please see the Package Explorer view or C/C++ Project view sections for more information.
