Step 3. Install updates for the ODE Build Feature

  1. In the Feature Updates view, expand the SDWB download site folder.
    Figure 8. Available SDWB features
  2. Select the ODE Build Feature:
    • Click Install Now > Next to install the feature updates.
    • If there is no Update Now push button in the Preview view, you have the latest updates.
    Figure 9. Update information for ODE Build Feature
  3. In the Feature License window, review and accept the license agreement, then click Next.
  4. In the Install Location window, click Finish.
  5. In the Feature Verification window, click Install to complete the installation.
  6. When prompted to restart the Workbench:
    • Click No if you are adding or updating other features.
    • Click Yes if you have completed all feature installs and updates.
