About this guide

The ODE Build Feature for Websphere Studio User Guide presents information about the Open Development Environment (ODE) Feature for WebSphere Studio. ODE is a development environment that enables common build processes across multiple operating systems. Other advantages of ODE include:

  • Provides dependency generation features
  • Simplifies makefiles
  • Standardizes makefiles for multiple platforms
  • Provides NLS support

The ODE Build Feature for WebSphere Studio enables you to perform the following ODE actions in an Eclipse environment:

  • Generate makefiles for Java and C/C++ projects
  • Build projects with ODE inside the Workbench
  • Build a pre-existing sandbox
  • View backing chain file listings
