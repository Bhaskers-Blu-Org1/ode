Supported ODE base application versions

The ODE Build Feature supports Version 4.0 or higher of the ODE base application. The required base application files are packaged with the Build Feature. You do not need to perform a separate install to get the base files.

Note:
The default value of the PATH environment variable is the location of the base files included with the Build Feature. To use your local version of the ODE base application, specify the path to your local version on the PATH environment variable.
